# API-Rest-NodeJS

Código fonte do curso realizado sobre a construçãos de APIs com NodeJS. No qual, no sistema é possivel realizar: 
- CRUD de produtos;
- Criação e recuperação de usuários;
- Criação de ordens de compra. 

Modulos utilizados: 

- HTTP 
- Express
- Debug
- Nodemon
- Body-parser
- Mongoose
- Guid
- JsonWebToken
